# 题目：玩会吧,别真学会了

## 题目描述

$\hspace{15px}$工作室某位学长非常内卷,在工作室呆到天荒地老,甚至因为晚归被锁进致新楼无法签到,可他仍不满足,他会每天挑选几个幸运数字录进他写的程序中,计算应该内卷多长时间。
$\hspace{15px}$你对此非常好奇,于是询问学长,学长对此解释称:程序会依据某种内部算法,在每对相邻数字之间随机插入加、减或异或运算符，并按照最后计算出所有可能出现的结果的总和作为自己额外的~~学习~~内卷时间。
$\hspace{15px}$此外,该程序还有特定的规则:

$\hspace{15px}$ 1.式子中要么仅有一个异或运算符,要么全是异或运算符。

$\hspace{15px}$ 2.当表达式中仅有一个异或运算符时，程序将分别计算其左右两侧所有可能值的总和，再将两个总和进行异或运算并返回结果。

$\hspace{15px}$ 3.加减优先级最高，其次是异或。

$\hspace{15px}$ 4.得出的结果可能很大,学长会累亖,因此结果对 $10^9+7$ 取余。

$\hspace{15px}$由于是内卷，具体的运算符组合以及中间过程不会对外公开。请你化身反内卷战士,通过代码计算学长每天内卷的时长。

## 输入格式

$\hspace{15px}$第一行输入一个整数 $n\,(2 \leq n \leq 10^5)$，表示学长幸运数字的个数。

$\hspace{15px}$第二行输入 $n$ 个非负整数$a_1$,$a_2$,...,$a_n\,(0 \leq a_i \leq 10^9)$,表示各个幸运数字。

## 输出格式

$\hspace{15px}$输出一个整数，表示所有可能的总和对 $10^9+7$ 取余后的结果。

## 样例输入

```text
3
0 2 5
```

## **样例输出**

```text
16
```
