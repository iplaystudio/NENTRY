# 题目：来幻觉了

## 题目描述

$\hspace{15px}$工作室有一个来自云南、特别爱吃蘑菇的学长。某天他吃蘑菇后出现了幻觉，说话带着乱七八糟的英语。他可能会把同一个单词说好几遍，而且单词里的字母顺序也被打乱了，搞得旁边的学长完全听不懂。现在给你他乱说的这些单词，请把字母组成相同的单词归成一组，并把每组的单词用空格隔开输出，帮学长弄清楚他到底在说什么。

![吃蘑菇](./9.jpg)

## 输入格式

$\hspace{15px}$输入一个正整数 $n\space(1 \leq n \leq 1000)$ 作为字符串数组的长度
$\hspace{15px}$接下来 $n$ 行,每行输入一个字符串 $s\space(2 \leq \left| s\right| \leq 10)$ 作为输入的单词，字符串中仅包含小写字母，保证所有字符串间互不相同。

## 输出格式

$\hspace{15px}$第一行输出一个正整数 $k$ 表示输出组数。
$\hspace{15px}$之后 $k$ 行，每一行代表一组字符串，组内的字符串之间用空格分隔。
$\hspace{15px}$你可以按任意顺序输出每个组和每个字符串，程序会自行判断答案。

## 样例输入

```text
8
eat
tea
tan
ate
nat
bat
abc
cab
```

## 样例输出

```text
eat tea ate
tan nat
bat
abc cab
```
