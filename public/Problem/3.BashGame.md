# 题目：黄瓜苦瓜之争

## 题目描述

Mutsumi 想种黄瓜，而 Mortis 想种苦瓜，但是因为月之森只分给了 Mutsumi 一块地，她们在种什么瓜的问题上发生了争执，所以爱玩石头的 Tomori 给她们出了一个主意——用石头一决胜负，玩法规则如下：
有 $m$ 个石头，每人每回合可以从其中拿走至少 $1$ 个、至多 $n$ 个石头，由 Mutsumi 先手，拿到最后一个石头的为胜者。

现在她们要进行 $T$ 局游戏，假设二人都足够聪明，每次总能做出最佳策略，请你提前判断出每局游戏的获胜者。

## 输入格式

在第一行输入一个整数 $T\,(1 \leq T \leq 100)$，表示二人进行的游戏局数。
接下来 $T$ 行，每行输入两个整数 $m$ 和 $n\, (1 \leq m,n \leq 10^{9})$
分别表示石头总数和每次最多可拿的石头数。

## 输出格式

输出共 $t$ 行，每行一个字符串。
若先手获胜输出 "Mutsumi Plants Cucumber"，否则输出 "Mortis Plants BitterMelon"（均不含引号）。

## 样例输入

```text
2
5 8
1000 9
```

## 样例输出

```text
Mutsumi Plants Cucumber
Mortis Plants BitterMelon
```
