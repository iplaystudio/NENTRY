# 题目：到底谁在学美工

## 题目描述

$\hspace{15px}$iPlay纳新啦!!!为此学长精心准备了宣传海报,由于学长不擅美工,制作过程十分艰难。
$\hspace{15px}$学长准备在海报上印一句"对称宣传语"--要求从给定的仅由英文字母和数字组成的字符串中找出其中最长的对称子串。如果有多个相同长度的最长对称子串，输出任意一个。
$\hspace{15px}$请你编写程序,帮学长打造吸睛海报。

## 输入格式

$\hspace{15px}$输入在一行中给出长度不超过 $10^3$ 的非空字符串,字符串中只包含大小写英文字母和数字。

## 输出格式

$\hspace{15px}$在一行中输出最长的对称子串（保持原字符串中的大小写形式）。

## 样例输入

```test
Welcomenewmemberstojoiniplaystudio
```

## 样例输出

```test
ene
```
