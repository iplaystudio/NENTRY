# 题目：异世界迷航

## 题目描述

你被泥头车撞了，但好消息是你穿越了,穿越到了 iPlay 的异世界,眼前浮现的是一段由字母与数字交织而成的乱码。想要回到现实，你必须从中 “破译” 出属于 iPlay 的密码信号。

从一个字符串中，提取出所有不区分大小写的 `I`、`P`、`L`、`A`、`Y` 字符，并将这些字符按照 `I → P → L → A → Y` 的顺序循环排列输出，但保留原始字符的大小写。
准确的说，你需要按顺序先输出一个 I 类字符，然后一个 P 类字符，然后一个 L 类字符，然后一个 A 类字符，然后一个 Y 类字符。

值得注意的是,这五类字符的出现次数可能并不相等，当某类字符用尽时，跳过该字符继续按顺序输出剩余字符，直到所有筛选出的字符都被输出。

## 输入格式

第一行输入一个正整数 $n\,(1 \leq n \leq 10^5)$ 代表字符串长度。
第二行输入一个长度为 $n$ 的、仅由英文字母和数字构成的字符串。

## 输出格式

对于每个测试用例，在一行中按题目要求输出排序后的字符串。题目保证输出非空。

## 样例输入

```text
15
3xI7pP8LlA4yYai
```

## 样例输出

```text
IpLAyiPlaY
```
