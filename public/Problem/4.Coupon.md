# 题目：外卖大战必修课

## 题目描述

依旧是工作室的某位学长，刚好在大学其间碰上了外卖大战，力度非常大，学长心动，学长行动。他点开了外卖平台，得到了一堆数额不同的优惠券，请你帮帮饿到眼花了看不清屏幕的学长高效地找到优惠力度最大的那张优惠券。

学长现在要点价格为 $p$ 的外卖，平台共给出了 $n$ 张形如满 $x$ 元减 $y$ 元的优惠卷，请你帮学长找出使用哪个优惠卷最划算，即用最低的价钱买到这份外卖。

![优惠券示意图](./4.png)
**(吃成小日本了说是)**

## 输入格式

第一行输入一个正整数 $n\,(1\leq n \leq 10^5)$ 和一个正整数 $p\,(1\leq p \leq 10^9)$ 分别代表优惠券数量和学长所点外卖的原价。

接下来 $n$ 行，每行输入一组正整数 $x$，$y\,(1\leq y \leq x \leq 10^9)$ ，输入格式为"$x$-$y$" ，代表该优惠券满 $x$ 元减 $y$ 元。

## 输出格式

最低价是多少,如果找不出来输出"nothing"(不含引号)

## 样例输入

```text
8
25
20-12
18-18
25-9
15-12
100-10
50-11
77-9
10-10
```

## 样例输出

```text
7
```
